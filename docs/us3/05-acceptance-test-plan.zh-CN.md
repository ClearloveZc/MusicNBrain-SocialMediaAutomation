## 1. 测试范围

验证 US3 MVP 是否满足 AC-1 到 AC-4 的验收标准。

## 2. 测试数据

1. 标题：Spring Student Recital
2. 时间：2026-03-15 19:00

## 4. 通过标准

US3 MVP 通过条件：

1. TC-01 到 TC-05 全部通过。
2. 至少一个负向场景（TC-06 或 TC-07）通过。
3. 不允许静默失败：所有失败任务都必须有 `error_code` 与 `error_message`。

## 5. 演示脚本（5-7 分钟）

1. 使用样例数据创建活动。
2. 生成内容并手动修改一条 caption。
3. 确认排程并入队。
4. 展示三平台任务状态，并执行一轮发布。
5. 演示一次失败场景与重试成功。
6. 展示最终成功/失败统计面板。
